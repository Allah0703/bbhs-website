Bangladesh Bio Health Society - Simple Website

Pages included:
1. index.html - Home page with only hero, about, contact details, and conference preview
2. membership.html - Membership page kept in the earlier simple form style
3. committee.html - Committee grid with placeholder members
4. conferences.html - Conference and seminar cards
5. journals.html - Microbial Systems plus two upcoming journal names
6. style.css - All design styles
7. script.js - Mobile menu, load more members, and static membership email draft

How to upload to GitHub Pages:
- Extract this ZIP file.
- Upload all files directly into the main/root folder of the repository.
- Make sure index.html is in the root folder.
- Commit changes and wait for GitHub Pages to update.

Membership form note:
The current form is static and opens an email draft to bangladeshbiohealthsociety@gmail.com.
For automatic online collection, connect it later with Google Forms, Formspree, Netlify Forms, or another form service.

Where to edit:
- Organization text: index.html
- Committee names/photos: committee.html
- Conference dates: conferences.html
- Journal titles/descriptions: journals.html
- Colors and spacing: style.css
