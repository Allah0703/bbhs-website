BBHS Simple Website v2

Files included:
- index.html
- about.html
- membership.html
- committee.html
- journals.html
- conferences.html
- resources.html
- contact.html
- style.css
- script.js

How to use:
1. Upload all files to your GitHub Pages repository.
2. Keep all files in the main/root folder unless you know how to change paths.
3. Open index.html to check the homepage.
4. Edit committee.html to replace "Name to be updated" with real member names.
5. Edit membership.html if you want to replace the Google Form link.
6. Edit journals.html if you want to add real journal links or cover images.
