# Bangladesh Bio Health Society Website

Static website starter for Bangladesh Bio Health Society (BBHS), ready for GitHub Pages.

## How to publish on GitHub Pages
1. Create a new GitHub repository, for example: `bbhs-website`.
2. Upload all files from this folder to the repository.
3. Go to Settings > Pages.
4. Under Build and deployment, select: Deploy from a branch.
5. Branch: `main`, folder: `/root`.
6. Save. The website will go live at: `https://YOUR-USERNAME.github.io/bbhs-website/`.

## Later custom domain
After buying `bangladeshbiohealthsociety.org`, add a `CNAME` file with:

bangladeshbiohealthsociety.org

Then update DNS records at the domain provider and enable HTTPS in GitHub Pages.
